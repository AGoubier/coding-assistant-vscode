# Browse the Catalog

The catalog organizes customizations from your configured sources into an easy-to-browse tree view.

## What You'll Find

- **Agents** -- AI agent configurations for Copilot, Claude, and other tools
- **Prompts** -- Reusable prompt templates and instructions
- **Skills** -- Specialized skill files that extend assistant capabilities
- **Rules** -- Project rules and coding guidelines

## How to Browse

1. Click the **Awesome Coding Assistants** icon in the Activity Bar (left sidebar)
2. Expand a source to see its folders and categories
3. Click any item to **preview** its contents
4. Use the install button to add items to your workspace

## Quick Actions

- **Search**: Use the search icon in the view toolbar to filter items
- **Install**: Click the download icon next to any item
- **Check for Updates**: Use the refresh icon to detect new and updated content
