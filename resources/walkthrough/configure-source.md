# Configure Your Source

Awesome Coding Assistants browses curated catalogs of AI coding assistant customizations -- agents, prompts, skills, and rules -- from one or more **index URLs**.

## What is an Index URL?

An index URL points to a JSON file that lists source repositories containing customizations you can browse, preview, and install. The default index points to the community catalog.

## How to Configure

1. Open **Settings** (`Ctrl+,` / `Cmd+,`)
2. Search for `awesome-coding-assistants.indexUrl`
3. Add or replace URLs in the array

You can add multiple URLs to merge catalogs from different sources, for example a community catalog and a private enterprise catalog.

### Setting Name

`awesome-coding-assistants.indexUrl`

> **Enterprise users**: Your administrator may have pre-configured this setting at the machine level. If so, the catalog is ready to browse -- no action needed.
